Terrain
In Edit: click hex to place units or paint terrain. In Play: click a friendly unit to show move (blue) + attack (red).

Activations: 3 per turn. Each unit may activate at most once per turn.
Command: within friendly GEN range (Green 3, Regular 4, Veteran 5) or friendly RUN relay range (1). Green out-of-command can't act; Regular out-of-command can't move.
Runner: RUN is always Green, has 2 HP, moves 4, has no attack, and relays command as a mini-general.
Engaged (adjacent) units can't move; SKR, RUN, and Veterans may disengage 1 hex to a hex not adjacent to any enemy.
Infantry pivot: in-command (or veteran) INF auto-pivots against flank/rear melee attacks before impact is resolved.
Line Advance: select friendly INF and press Line Advance (or L) to push a contiguous line forward 1 hex as one activation (movement only, partial if blocked; line shape follows Advance axis). Selected INF now previews the exact line and forward direction before commit.
Veteran cavalry: after attacking in melee, a veteran CAV may disengage 1 hex if it has movement remaining and can end not adjacent to enemies.

Online: set Game mode to Online, Host creates room and shares a 4-character code, friend enters code to connect. Host controls setup and mode; Host plays Blue, Guest plays Red.
Dice: 5–6 = Hit, 4 = Retreat (blocked retreat → 1 hit). Woods: defender reduces attacker dice by 1 (min 1).
Pass: press P with a selected unit.
UI: spent units are dim. Green out-of-command units show a dashed orange ring. Toggle command radius with C.
Reference

Bannerfall Rules

Turn And Activations

Each side gets 3 activations per turn.

On activation, a unit may move, attack, or pass. Line Advance is a formation order for INF (movement-only).

A unit normally activates at most once per turn. End Turn swaps side, increments turn after Red, and resets activations/spent units.

Unit Stats (MP / HP / UP)

INF
MP 1 · HP G/R/V 3/4/5 · UP G/R/V 3/5/7
CAV
MP 2 · HP G/R/V 2/3/4 · UP G/R/V 6/8/10
SKR
MP 2 · HP G/R/V 1/2/3 · UP G/R/V 2/3/4
ARC
MP 1 · HP G/R/V 1/2/3 · UP G/R/V 2/4/6
GEN
MP 2 · HP G/R/V 2/3/4 · UP G/R/V 8/10/12
RUN
MP 4 · HP 2 · UP 1 · no attack

RUN is always Green and works as a command relay piece.

Command Network

General command radii: Green 3, Regular 4, Veteran 5.

Runner relay command radius: 1 hex. RUN has no attack and uses the RUN stats shown above.

Command-independent units: GEN, RUN, CAV, and all Veterans.

Green command-dependent units out of command cannot activate. Regular command-dependent units out of command may attack from current hex but cannot move.

Movement And Engagement

If adjacent to an enemy, a unit is engaged and usually cannot move.

SKR, RUN, and Veterans may disengage 1 hex, but must end on a hex not adjacent to enemies.

Clear
Entry cost 1 for all unit types.
Hills/Woods/Rough
Entry cost 2 for INF/ARC/SKR/GEN/RUN, and 3 for CAV.
Water
Impassable (cannot enter).

Line Advance: select a friendly INF; the game previews contiguous eligible INF in line and forward destination arrows. Resolve as one activation, movement only, partial if blocked; all participants become spent.

Combat Resolution

Each d6 result: 5–6 = Hit (−1 HP), 4 = Retreat, 1–3 = No effect.

Melee Dice
INF 2, CAV 3, SKR 2, ARC 1, GEN 1, RUN 0
Ranged Dice
ARC: range 2 (2 dice), range 3 (1 die). SKR: range 2 only (1 die).
Ranged Limits
Only ARC/SKR have ranged attacks; engaged units cannot make ranged attacks.
Woods Defense
Attacker rolls −1 die (minimum 1).
Retreat Rule
Retreat pushes defender to a neighboring hex that increases distance from attacker. If retreat hex is off-board, water, or occupied, that retreat converts to 1 hit.
Retreat Chain
Multiple retreat results resolve one step at a time; each step can retreat or convert to hit.

Victory Modes

Annihilation: destroy all enemy units (including generals).

Decapitation: destroy all enemy generals.

Clear Victory (UP): capture enemy UP at least equal to ceil(enemy starting UP / 2). UP is gained when an enemy unit is destroyed.

Controls And Hotkeys

Hotkeys: P pass, L line advance, C command overlay, T terrain palette, Esc close rules.

Online mode: Host controls setup/rules/start, plays Blue. Guest joins as Red.
