Terrain
In Edit: click hex to place units or paint terrain. In Play: click a friendly unit to show move (blue) + attack (red).

Activations: 3 per turn. Each unit may activate at most once per turn.
Command: within friendly GEN range (Green 3, Regular 4, Veteran 5) or friendly RUN relay range (1). Green out-of-command can't act; Regular out-of-command can't move.
Runner: RUN is always Green, has 2 HP, moves 4, has no attack, and relays command as a mini-general.
Engaged (adjacent) units can't move; SKR, RUN, and Veterans may disengage 1 hex to a hex not adjacent to any enemy.
Infantry pivot: in-command (or veteran) INF auto-pivots against flank/rear melee attacks before impact is resolved.
Line Advance: select friendly INF and press Line Advance (or L) to push a contiguous line forward 1 hex as one activation (movement only, partial if blocked; line shape follows Advance axis).
Veteran cavalry: after attacking in melee, a veteran CAV may disengage 1 hex if it has movement remaining and can end not adjacent to enemies.

Online: set Game mode to Online, Host creates room and shares a 4-character code, friend enters code to connect. Host controls setup and mode; Host plays Blue, Guest plays Red.
Dice: 5–6 = Hit, 4 = Retreat (blocked retreat → 1 hit). Woods: defender reduces attacker dice by 1 (min 1).
Pass: press P with a selected unit.
UI: spent units are dim. Green out-of-command units show a dashed orange ring. Toggle command radius with C.
Reference

Bannerfall Rules

Turn Structure

Each side gets 3 activations per turn. A unit usually acts once per turn.

On activation: move, attack, or pass. Line Advance is a movement-only formation order.

Command Network

General command radii: Green 3, Regular 4, Veteran 5.

Runner relay: moves 4, command radius 1, HP 2, UP 1, no attack.

Green command-dependent units out of command cannot activate. Regular command-dependent units out of command can attack but cannot move.

Movement And Engagement

If adjacent to an enemy, a unit is engaged and usually cannot move.

SKR, RUN, and Veterans may disengage 1 hex to a hex not adjacent to any enemy.

Terrain: Clear cost 1. Hills/Woods/Rough cost 2 (CAV cost 3). Water is impassable.

Combat

Die results: 5–6 hit, 4 retreat, 1–3 no effect. Blocked retreat converts to 1 hit.

Melee Dice
INF 2, CAV 3, SKR 2, ARC 1, GEN 1, RUN 0
Ranged
ARC: range 2 (2 dice), range 3 (1 die). SKR: range 2 (1 die).
Woods Defense
Attacker rolls −1 die (minimum 1).

Victory And Controls

Victory modes: Annihilation, Decapitation, Clear Victory (capture at least half enemy starting UP).

Hotkeys: P pass, L line advance, C command overlay, T terrain palette, Esc close rules.
